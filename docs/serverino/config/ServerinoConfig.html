    Struct ServerinoConfig

    Struct used to setup serverino. You must return this struct from a function with @onServerInit UDA attached.

    struct ServerinoConfig ; 
    @onServerInit
auto configure()
{
   // You can chain methods
   ServerinoConfig config =
      ServerinoConfig.create()
      .setWorkers(5)
      .enableKeepAlive();

   return config;
}
